# 비개발자 AI 해커톤 플랫폼 제품 로드맵

## 로드맵 개요

### 전체 타임라인
```
Phase 1 (MLP): 3개월 - 기본 해커톤 운영 가능
Phase 2 (UX Enhancement): 2개월 - 사용자 경험 고도화
Phase 3 (Enterprise): 3개월 - 기업 특화 기능
Phase 4 (Global): 6개월 - 글로벌 확장 기능
```

### 핵심 가설 검증 순서
1. **비개발자도 AI로 유의미한 결과물을 만들 수 있는가?**
2. **교육-해커톤 통합 플로우가 효과적인가?**
3. **기업 HRD팀이 기존 교육 대신 이것을 선택할까?**
4. **해커톤 단위 과금 모델이 지속 가능한가?**

## Phase 1: MLP (Minimum Lovable Product)
*목표: 해커톤 1회를 처음부터 끝까지 운영 가능한 제품*

### 타임라인: 3개월 (2026.04 - 2026.06)

### 핵심 기능

#### 1.1 해커톤 생성 및 관리 (2주)
**운영자 기능**
- [ ] 해커톤 기본 정보 설정 (제목, 일시, 참가인원, 주제)
- [ ] 참가 링크 생성 및 공유
- [ ] 해커톤 시작/종료/일시정지 원클릭 제어
- [ ] 제한 시간 설정 및 타이머 표시

**기술 스택**
- Backend: Node.js + Express + PostgreSQL
- Frontend: React + TypeScript
- 인증: JWT 기반 세션 관리
- 배포: AWS EC2 + RDS

#### 1.2 교육 모드 (4주)
**가이드형 튜토리얼**
- [ ] 단계별 학습 콘텐츠 (10단계)
  - Step 1: AI와의 첫 대화
  - Step 2: 간단한 웹페이지 요청
  - Step 3: 스타일링 추가
  - Step 4: 인터랙션 추가
  - Step 5: 데이터 처리
  - Step 6: API 연동
  - Step 7: 배포
  - Step 8: 디버깅
  - Step 9: 개선 및 반복
  - Step 10: 최종 점검

**AI 튜터 시스템**
- [ ] GPT-4 기반 단계별 힌트 제공
- [ ] 진행 상황에 따른 맞춤형 가이드
- [ ] 오류 해결 도우미
- [ ] 진도 체크 및 다음 단계 안내

#### 1.3 웹 기반 AI 에디터 (6주)

### 기술 아키텍처

#### 시스템 구성도
```
Frontend (React)
    ↓
API Gateway (Express)
    ↓
┌─────────────┬─────────────┬─────────────┐
│ Auth Service│ Editor Service│ AI Service  │
└─────────────┴─────────────┴─────────────┘
    ↓
PostgreSQL Database
    ↓
File Storage (S3)
```

#### API 설계
```
POST /api/hackathons - 해커톤 생성
GET /api/hackathons/:id - 해커톤 정보 조회
POST /api/hackathons/:id/start - 해커톤 시작
POST /api/hackathons/:id/end - 해커톤 종료

POST /api/ai/chat - AI 대화
POST /api/ai/code - 코드 생성 요청
POST /api/editor/save - 코드 저장
POST /api/editor/preview - 미리보기 생성

POST /api/submissions - 결과물 제출
GET /api/submissions/:hackathonId - 결과물 목록
POST /api/evaluations - 평가 제출
```

### 성공 지표
- [ ] 해커톤 완주율 70% 이상
- [ ] 교육 모드 완료율 80% 이상
- [ ] 결과물 품질 점수 평균 3.5/5 이상
- [ ] 시스템 가동률 99% 이상
- [ ] 파일럿 고객 만족도 4/5 이상

### 리스크 및 대응
